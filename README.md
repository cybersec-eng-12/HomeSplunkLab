# Enterprise SIEM Implementation with Splunk & Active Directory
Practical homelab of Windows VMs connected together via Active Directory Domain. Splunk is used as a SIEM for this network. Kali Linux is used to simulate attacks. 

# Objectives
- Simulate an enterprise SOC environment to practice log ingestion, threat detection, incident response, & penetration testing.
- Practice Splunk queries.
- Practice setting up & connecting computers to an on-premise Active Directory Domain Controller.

# Tools Used
- Splunk Enterprise
- Sysmon
- Windows Server 2022
- Active Directory
- Oracle VirtualBox
- Kali Linux
- Windows 11 Pro
- Ubuntu Server 24
